# 📅 Smart Timetable System

A web-based application designed to automate and manage timetable scheduling efficiently for educational institutions. This system reduces manual effort and avoids scheduling conflicts using intelligent allocation.

---

## 🚀 Features

* 📌 Automated timetable generation
* 👩‍🏫 Lecturer and subject management
* 🕒 Time slot allocation
* 🏫 Class and section handling
* ⚡ Conflict-free scheduling
* 📊 Data handling using SQLite
* 🌐 User-friendly interface

---

## 🛠️ Tech Stack

* **Backend:** Python (Flask)
* **Database:** SQLite
* **Frontend:** HTML, CSS
* **Libraries:** Pandas, SQLAlchemy, Flask-CORS

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```
git clone https://github.com/Vamshi10052005/Smart-Timetable-system.git
cd Smart-Timetable-system
```

### 2️⃣ Create virtual environment

```
python -m venv venv
venv\Scripts\activate
```

### 3️⃣ Install dependencies

```
pip install -r requirements.txt
```

### 4️⃣ Run the application

```
python app.py
```
